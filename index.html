Bem-vindo ao Crocreator!

O editor de croquis de escalada mais completo do Brasil, criado com carinho para a comunidade da escalada.

Em menos de 1 minuto, veja como tudo funciona.

Tela escura = modo noturno
Grid = 2m por célula
Régua lateral = altitude em metros
🛠️

Ferramentas

A barra superior tem tudo que você precisa para criar seu croqui.

V·Selecionar — Mover, redimensionar e rotacionar
P·Caneta — Desenhar linhas e traçados
T·Texto — Escrever notas no croqui
L·Travar / Destravar item selecionado
Del·Excluir selecionados
Ctrl+Z·Desfazer
+ / -·Redimensionar
[ / ]·Rotacionar 15°
📚

Painel de Símbolos

O painel lateral esquerdo contém todos os símbolos organizados por categoria. Clique em um símbolo para inserir no canvas, ou arraste diretamente para o local desejado.

Proteção Fixa
Parada Fixa
Reunião
Diedro
Atenção
Fissura
📸

Imagem de Fundo

Use o botão de foto 🖼 na barra para carregar uma imagem da parede de rocha como guia.

Arraste·Posicione a imagem no local certo
+ / -·Redimensione a foto
Opacidade·Ajuste via slider na barra superior
P/B·Converta para preto e branco
L·Trave a foto para não mover acidentalmente
📤

Exportar o Croqui

Quando o croqui estiver pronto, exporte diretamente pela barra superior.

SVG·Arquivo vetorial, editável no Illustrator / Inkscape
PDF·Alta qualidade, pronto para impressão
